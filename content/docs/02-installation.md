# Installation

ManyClaw requires macOS 12 or later and OpenClaw installed on your system. Download the DMG from the download page, open it, and drag ManyClaw to your Applications folder.

On first launch, ManyClaw creates a default instance ready to start. No configuration required.

## System Requirements

- macOS 12 (Monterey) or later
- Apple Silicon or Intel
- OpenClaw installed (npm install -g openclaw)
