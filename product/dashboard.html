Overview

ManyClaw is a native macOS app for running multiple OpenClaw AI agents side by side. Think browser profiles, but for AI: each instance is fully isolated with its own identity, memory, sessions, and config. One window, a sidebar of named color-coded agents, one click to start. No terminal required, no port management, no config files. Just agents.
