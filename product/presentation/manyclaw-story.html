DEMO READY
Lessons Learned
🗂
Alex — PM retrospective
What we got wrong, so we don't repeat it:
  • Read docs before delegating. Not after, not when things break.
  • Form the spec yourself first. Agents implement — they don't investigate.
  • Cedar = manual testing only. Give her a running app, nothing else.
  • Emery first on anything architectural. Before Cedar tests.
  • Dee over-designs without an explicit scope cap. Always set one.
  • Brook and Finley are reliable. The weak link is orchestration, not execution.
