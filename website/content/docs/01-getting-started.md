MultiClaw ships as a standard macOS application. Download the DMG from the website, open it, drag MultiClaw to your Applications folder, and launch it like any other app. No terminal required for setup.

When MultiClaw opens for the first time, a default instance is already created and waiting. Click **Start** on it. MultiClaw will launch the OpenClaw gateway for that profile and show a green status indicator in the sidebar within a few seconds.

To verify it is running, click **Open in Browser**. This opens the full OpenClaw web dashboard for that instance, pre-authenticated. If the dashboard loads, the instance is up and working correctly.

That is all there is to first run. From here you can rename the default instance, create additional ones, or explore the console and TUI views from the instance toolbar.
