# Running Multiple Instances

Click + in the sidebar header to create a new instance. Give it a name and pick a colour — these are yours to organise however makes sense. Dev and Production. Personal and Work. Fast and careful.

Each instance is independent. Start and stop them individually. Drag to reorder. The sidebar shows live status — green is running, grey is stopped.

## The TUI

Each running instance has a terminal view. Click TUI in the instance toolbar to switch from the log console to an interactive terminal connected to that agent.
